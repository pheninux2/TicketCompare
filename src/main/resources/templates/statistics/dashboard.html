Comment fonctionnent les Statistiques ?
Méthode : Analyse Agrégée

Le système analyse tous vos tickets pour calculer des statistiques par catégorie de produits.

Données Calculées
  • Prix Moyen : Moyenne de tous les prix pour un produit ou catégorie
  • Prix Maximum : Le prix le plus élevé observé
  • Prix Minimum : Le prix le plus bas observé
  • Nombre de Produits : Quantité totale de produits dans la catégorie
Limitations
  • Les statistiques reflètent uniquement vos achats personnels
  • Plus vous avez de tickets, plus les statistiques sont représentatives
  • Les produits sans catégorie apparaissent dans "Autre"
  • Les prix varient selon les magasins et les dates d'achat
